Net-Shop
Plataforma E-Commerce

Proyecto Full Stack desarrollado con Java, Spring Boot, MySQL, Bootstrap y API REST para gestionar una tienda en línea con autenticación de usuarios, carrito de compras y catálogo dinámico.

Características principales

  • Registro e inicio de sesión de usuarios
  • Panel de administración de productos
  • Carrito de compras y proceso de pago
  • Consumo de API REST para integración de servicios
  • Diseño responsivo con Bootstrap

Tecnologías utilizadas

  • Backend: Java, Spring Boot, Spring Security
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Base de datos: MySQL
  • Control de versiones: Git y GitHub
